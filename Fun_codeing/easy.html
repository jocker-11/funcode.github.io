  • 首先，在 int 那邊加上 y 變數
  • 然後回到 if 的 { } 裡
  • 如果前者數值>後者數值，也就是 box[n] > box[n+1] 時，那我就必須讓 box[n] 和 box[n+1] 裡的數值都先取出來， 再把 box[n] 的放進 box[n+1] 裡、把 box[n+1] 的放進 box[n] 裡。
  • 這時就會用到 y 了，你把 y 想成是一個暫時借放的暫存器，你可以把從 box 裡取出的數值借放到 y 裡面
  • 先把 box[n+1] 的值借放進 y 裡 >> y=box[n+1]
  • 把 box[n] 的值放進 box[n+1] 裡 >> box[n+1]=box[n] ;
  • 最後，再把借放在 y 裡的數值放入 box[n] 裡 >> box[n]=y ;
